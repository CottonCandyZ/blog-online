## 感谢

首先是感谢下面列出的项目，在制作这个 Blog 时~~抄~~参考了以下的项目，他们的 Blog 都非常好看：

- [DouLog](https://im.daidr.me/) 受此启发想要自己制作 Blog，因为动画很酷。参照其制作了桌面版的导航按钮。Blog 的最初版本也是参照其制作的，但后来觉得还是优先展现内容比较好（原来有一个启动卡片）。

- [fuwari](https://github.com/saicaca/fuwari) 在 [V2](https://www.v2ex.com/t/1032293#r_14583587) 上看到有小伙伴分享，觉得非常好看。原本网站的基础样式实际都是参照 JoshWComeau 做的，但是更喜欢这种模块化的风格，遂参考在近期重建。

另外 Blog 右侧的头像的画师为 [lovelivexAx](https://twitter.com/lovelivexAx)。

Code 的话，主要是喜欢这种交互的感觉，像是在搭积木，解决一个又一个问题会有成就感，正如我在[关于这个 Blog](posts/2024-03-12-about-this-blog)里所说，这里是我初次自己尝试建造一个 Blog，它可能会有一些问题，但是我应该会不断的优化，加入一些我学到的新东西，添加一些动画和效果，整个过程应该会非常有趣。

[前身](https://cottoncandyz.notion.site/)，之前会将一些日常或者笔记整理在 Notion 上，由于它可以直接像操纵数据库一样操纵表格，包括外键啥的，是一个很方便的 All in One App，所以在没有这个 Blog 之前，一些平时写的东西基本都存在上面，以后应该会把一些比较重的内容仍然放在上面，包括照片墙和笔记。
