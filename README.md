# 棉花糖的 Blog

使用 Next.js 创建，TS & Tailwind，Responsive design。

项目实际上使用 t3 创建的，存有 tRPC 和 Prisma 的配置，主要是为了后期可以转移存储的方式和添加一些互动元素，比如评论区。
